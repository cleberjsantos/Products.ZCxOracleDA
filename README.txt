What is ZCxOracleDA?
----------------------

ZCxOracleDA is both a tiny frontend to the Cx_Oracle package for Zope 2 and Zope
3. In addition it acts as a database adapter for ZSQL methods and supports Oracle databases only.


Requirements:
--------------

  - Zope 2.8 +

  - cx_Oracle 5.1.2 +
